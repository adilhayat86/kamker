# KAMKER MASTER PRD

Version 1.0

## Vision
Kamker is a mobile-first marketplace that connects customers directly with local professionals without acting as a middleman.

## Business Model
Revenue sources:
- Advertising
- Featured listings
- Verification fees
- Premium subscriptions
- Paid requirement broadcasts

## Core Categories
- Nurses
- Maids
- Drivers
- Teachers
- Handwriting Teachers
- Tutors
- Electricians
- Plumbers
- Cooks
- Guards
- Beauticians
- Artists

## Core Features
1. Professional Directory
2. Professional Profiles
3. Direct Call and WhatsApp Contact
4. Send Requirement
5. Admin Dashboard
6. Verification System
7. Paid Broadcast System

## Customer Flow
Browse category → View professional → Contact directly or Send Requirement.

## Professional Flow
Register → Verification → Profile Active → Receive Leads.

## Send Requirement
Store customer requirements and later broadcast them to matching professionals.

## Technology
- Next.js
- Supabase
- PostgreSQL
- Vercel

## Roadmap
Phase 1: Directory
Phase 2: Requirements
Phase 3: Broadcasts
Phase 4: Payments
Phase 5: Mobile Apps

## Goal
Become Pakistan's largest local professional directory platform.
