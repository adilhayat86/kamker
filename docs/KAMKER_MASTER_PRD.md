# KAMKER MASTER PRD

Version 2.0

# Executive Summary
Kamker is a Pakistan-focused marketplace for local professionals. The platform connects customers directly with professionals and avoids the traditional middleman model.

# Vision
Build the largest searchable directory of trusted professionals in Pakistan.

# Mission
Help every household and business find local talent within minutes.

# Core Principles
- Direct contact
- No commission on jobs
- Mobile first
- Fast search
- Simple user experience
- Scalable nationwide directory

# User Types
## Customers
People looking for services.

## Professionals
People offering services.

## Administrators
Manage listings, requirements and platform quality.

# Customer Journey
1. Visit Kamker
2. Search category
3. Filter by city
4. View profile
5. Call or WhatsApp professional
6. Alternatively submit a requirement

# Professional Journey
1. Register
2. Add profile details
3. Verify phone
4. Optional CNIC verification
5. Receive leads and enquiries

# Send Requirement System
Fields:
- Service
- City
- Area
- Budget
- Urgency
- Phone
- WhatsApp
- Description

Requirements are stored and later matched to professionals.

# Paid Broadcasts
Customers can pay to send a requirement to all matching professionals in a city and category.
